# GenStack-Zero

Python project scaffold for GenStack-Zero.

## Project Structure

```text
src/        Core Python package and source code
apps/       Application entry points and runnable apps
data/       Local datasets and generated data artifacts
notebooks/  Exploratory notebooks
configs/    Configuration files
tests/      Automated tests
```

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Testing

```bash
pytest
```
